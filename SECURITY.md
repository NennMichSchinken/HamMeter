# Security Policy

HamMeter is a small hobby project, but if you find a security issue, please report it responsibly.

## Reporting a Vulnerability

Please use GitHub's [private vulnerability reporting](https://github.com/NennMichSchinken/HamMeter/security/advisories/new) instead of opening a public issue. This lets us discuss and fix the problem before it's publicly visible.

We'll do our best to respond and get a fix out in a reasonable time, given this is maintained in spare time.

## Supported Versions

Only the latest released version of HamMeter is supported. Please make sure you're on the current version before reporting an issue.
